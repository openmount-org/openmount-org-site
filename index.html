OPEN-MOUNT.ORG

Astronómia al alcance de todos

Misión

Open-Mount.org es una iniciativa de código abierto dedicada al diseño y desarrollo de monturas para telescopios de alta precisión. Buscamos eliminar las barreras económicas en la astronomía amateur proporcionando planos, software y guías técnicas de libre acceso.

Este proyecto se basa y apoya en el ecosistema OnStep, la plataforma de control creada por Howard Dutton. Sin la que seria imposible realizar este proyecto

OnStep proporciona el software, nuestra intención es poner a disposición el hardware

Colaboración

Al ser un proyecto de todos y para todos, aceptamos y agradecemos cualquier sugerencia, aportación o mejora de la comunidad. Si eres ingeniero, astrónomo o entusiasta, tu conocimiento es bienvenido para evolucionar este diseño.
